About the Project

This project shows a simple way of doing sentiment analysis in your web-browser with the help of vaderSentiment tool. This web-application is mainly suitable(and made) for memes and texts that appear in social-media bodies. The motivation for this project originates from the fact that there exist no such application which could analyse the sentiment from memes. Though memes being highly contextual and sarcastic, we believe the vaderSentiment tool is of great help in this regard.
